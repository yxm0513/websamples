    Things To Know

    • GLOBAL: Declared outside of a function. It exists on the web page, available until the page is closed
    • LOCAL: Declared inside a function. It exists when the function starts, and is done when the function returns something, ie: ends
    • Dev Tools. Keep them open always when writing JavaScript
    • console.log - allows us to execute JS and see it in the browser

    PART 2: Advanced JS - Closures

    • Closures change the accessibility of private variables to 'free' variables
    • Closure is a function that has access to an outer (higher level scope) function
    • It's the ability to retain state and scope after the function returns

    When in doubt about scope of a variable or function, check your context!
